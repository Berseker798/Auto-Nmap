AutoNmap
[PL] Opis projektu
AutoNmap to automatyczne narzędzie CLI (konsolowe) napisane w Pythonie i skompilowane do pliku wykonywalnego .exe. Służy do szybkiego skanowania sieci i wykrywania otwartych portów oraz usług na urządzeniach. Automatyzuje proces korzystania z popularnego skanera Nmap, ułatwiając pracę podczas testów bezpieczeństwa we własnym laboratorium.

Wymagania:
System operacyjny Windows.

Zainstalowany program Nmap w systemie (narzędzie musi mieć dostęp do komendy nmap w konsoli).

Jak używać:
Uruchom plik AutoNmap.exe przez wiersz poleceń (CMD / PowerShell).

Podaj adres IP lub zakres sieci, który chcesz przetestować.

Narzędzie automatycznie dobierze parametry skanowania i wyświetli wynik.

[EN] Project Description
AutoNmap is an automated CLI tool written in Python and compiled into a standalone .exe executable. It is designed for quick network scanning and discovering open ports and services on devices. It automates the process of using the popular Nmap scanner, making security testing in your own lab much easier.

Requirements:
Windows OS.

Nmap installed on the system (the tool requires the nmap command to be available in the system PATH).

How to use:
Run AutoNmap.exe via Command Prompt (CMD) or PowerShell.

Enter the IP address or network range you want to test.

The tool will automatically configure the scan parameters and display the results.

[DE] Projektbeschreibung
AutoNmap ist ein automatisiertes CLI-Tool, das in Python geschrieben und in eine ausführbare .exe-Datei kompiliert wurde. Es dient zum schnellen Scannen von Netzwerken und zum Erkennen offener Ports und Dienste auf Geräten. Es automatisiert die Nutzung des beliebten Nmap-Scanners und erleichtert Sicherheitsanalysen im eigenen Labor.

Anforderungen:
Windows-Betriebssystem.

Installiertes Nmap im System (das Tool erfordert, dass der Befehl nmap in der Konsole verfügbar ist).

Verwendung:
Führen Sie die Datei AutoNmap.exe über die Eingabeaufforderung (CMD) oder die PowerShell aus.

Geben Sie die IP-Adresse oder den Netzwerkbereich ein, den Sie testen möchten.

Das Tool wählt automatisch die Scan-Parameter aus und zeigt das Ergebnis an.
